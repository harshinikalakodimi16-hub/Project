# Project Flow

## Overview

The EduGenie project follows a simple workflow to provide AI-powered educational assistance using Google Gemini.

### Workflow

1. User opens the EduGenie website.
2. User enters a learning question.
3. The frontend sends the request to the Google Gemini API.
4. Gemini processes the question and generates an answer.
5. The response is displayed on the website.
6. The interaction is recorded as part of the learning session.
