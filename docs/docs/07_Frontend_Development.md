# Frontend Development

## Overview

The frontend of EduGenie is developed using HTML, CSS, and JavaScript to provide a clean, responsive, and user-friendly interface.

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Features

* Responsive design
* Easy navigation
* Chat interface
* User-friendly layout
* Mobile and desktop compatible

## User Interface

The interface allows users to enter educational questions and receive AI-generated responses instantly through an interactive chat experience.
