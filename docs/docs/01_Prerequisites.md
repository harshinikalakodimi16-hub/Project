# Prerequisites

## Overview

Before developing EduGenie – Google Gemini Powered Learning Assistant, the following software, tools, and technologies were required.

## Hardware Requirements

* Computer or Laptop
* Minimum 4 GB RAM
* Stable Internet Connection

## Software Requirements

* Visual Studio Code
* Git
* GitHub
* Modern Web Browser (Google Chrome or Microsoft Edge)
* Google AI Studio (for Gemini API)
* GitHub Pages

## Programming Languages

* HTML5
* CSS3
* JavaScript

## APIs and Technologies

* Google Gemini API
* GitHub Pages for deployment

## Development Environment

The project was developed using Visual Studio Code and deployed using GitHub Pages. Google Gemini API was integrated to provide AI-powered educational responses.
